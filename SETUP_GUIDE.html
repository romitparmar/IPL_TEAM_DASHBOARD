IPL Dashboard — Setup Guide
NoSQL Micro Project  ·  MongoDB + Flask  ·  Step-by-step
MongoDB Flask (Python)
STEP 01
Install Required Software

You need Python and MongoDB installed on your system. Download them from their official sites if not already installed.

MongoDB: Download from mongodb.com/try/download/community

Python 3.x: Download from python.org/downloads

STEP 02
Set Up Project Folder

Create a folder and set up the structure like this:

ipl-dashboard/ ├── app.py ├── requirements.txt └── templates/ └── index.html

Place the files you received (app.py, requirements.txt, index.html) exactly as shown above.

STEP 03
Create requirements.txt

Create a new file called requirements.txt in your project folder and paste this inside:

flask flask-cors pymongo

Then open your terminal, go inside the project folder, and run:

cd ipl-dashboard pip install -r requirements.txt

This installs: Flask, PyMongo, and Flask-CORS.

STEP 04
Start MongoDB

Open a new terminal window and start MongoDB:

On Windows:

net start MongoDB

On Mac/Linux:

mongod
MongoDB runs on port 27017 by default. Keep this terminal open while using the app!
STEP 05
Run the Flask App

In your original terminal (inside the project folder), run:

python app.py

You should see output like:

Seeded IPL 2024 teams Seeded IPL 2024 players * Running on http://127.0.0.1:5000
Open your browser and go to: http://localhost:5000 — your IPL Dashboard is live!
DATABASE SCHEMA
MongoDB Collections

The app automatically creates these 2 collections in a database called ipl_dashboard with real IPL 2024 data:

Collection: players

FieldTypeDescription
_idObjectIdAuto-generated by MongoDB
nameStringPlayer full name (e.g. Virat Kohli)
teamStringTeam code (MI, CSK, RCB, KKR...)
roleStringBatsman / Bowler / All-rounder / Wicket-keeper
matchesIntMatches played in IPL 2024
runsIntTotal runs scored in IPL 2024
avgFloatBatting average
srFloatStrike rate

Collection: teams

FieldTypeDescription
_idObjectIdAuto-generated by MongoDB
teamStringTeam short code (e.g. KKR, SRH)
full_nameStringFull team name (e.g. Kolkata Knight Riders)
winsIntNumber of wins in IPL 2024
lossesIntNumber of losses in IPL 2024
pointsIntTotal points (wins × 2)
matchesIntTotal matches played
nrrFloatNet Run Rate (real IPL 2024 value)
All data is seeded automatically on first run. No manual entry needed — 10 IPL 2024 teams and 22 real players are inserted into MongoDB when you start the app for the first time.
QUICK REFERENCE
Useful Commands

Verify MongoDB data in MongoDB shell:

mongosh use ipl_dashboard db.teams.find().pretty() db.players.find().pretty()

If you want to reset and re-seed all data fresh:

mongosh use ipl_dashboard db.players.drop() db.teams.drop() exit

Then run python app.py again — it will re-seed everything automatically.
